目前，我在南昌大学（211 工程高校）数学与计算机学院攻读硕士研究生，并在计算中心实验室学习，师从陈炼教授，期间曾获得研究生学业奖学金，“华为杯”第二十届中国研究生数学建模竞赛全国二等奖。此前，我在本科的时候在微尺度网络计算实验室师从年近 60 岁拥有丰富经验的王雪飞老师学习计算机技术，他是一位杰出且富有情怀的学者和精通软硬件的全栈工程师，给予我最初的指导和示范，铭记在心。最敬佩的人是吴军老师（Google 高级研究员，腾讯副总裁），他的书籍，专栏和课程让我在计算机科学，数学，文学和生活等方面的认知上获得很大的提升，接受了他精神上的洗礼。本科期间我的精神榜样是李苦李老师（华为架构师） ，感谢他的文章和关键时刻的指导帮助我克服了自学和考研路上遇到的困难，帮助了我成长，让我学会了和时间做朋友。大学过后一直远程和钟辉老师（阿里云后端专家，AI 策略负责人）学习，每周的直播周会和文档的耳濡目染，让我学到很多计算机行业学习和做事方法论。尤其感谢表哥李岩松（华为安全研究员）鼓励我勇敢的迈出了自学计算机的第一步，并在后来在学习上提供的鼓励和帮助。在最后我还想感谢我人生中遇到的所有曽给我直接或间接表达过善意的人，爸妈，姥姥姥爷，学校的老师，师兄，数不清的同学，网络上的陌生人，书籍中没有见过面的作者……

我能够熟练使用 Java，Python，C，JavaScript，HTML 和 CSS 等语言，了解 JVM 原理，设计模式，UML，良好的软件工程素养。熟练使用 MySQL，了解原理。熟练使用 Spring，Mybatis，Redis，Kafka 等框架和中间件技术，了解原理，了解分布式理论与技术。熟练使用 IDEA，Git 等开发工具。计算机基础扎实，在课余时间学习了数据分析和人工智能等相关领域的知识技能，以期望在未来获得更多的可能。我经常会尝试各种学习工具帮助我提高效率，不断的优化自己的学习方式，专注力，比如 XMind，幕布，语雀文档整理笔记沉淀知识，番茄钟，ChatGPT 辅助学习，熟悉 Prompt 工程，理解大模型的工作原理（NLP，ML）。使用清单规划每日的安排，每个年度，季度的 OKR 计划，坚持写周报，日报，不断的反思挑战自我规划未来。

我来自大别山的一个红色小镇，小时候没有认识到学习的重要性，中学在一所乡镇高中考进入一所公办大学学习，并继续通过努力考上 211 工程高校研究生。目前专注于计算机互联网领域，同时也对系统控制，电子，电气，机械领域都有所了解，喜欢羽毛球（院羽毛球队队员，曽获得院级比赛前三名的成绩），跑步，骑行（曽单程骑行 300 公里），游泳（曽接受过专业游泳培训），健身，听音乐安静的思考，读书以获得很多不同的看问题的方式、知识、技能和经验，看纪录片，旅行，积极参与各种社交活动获得很多对这个世界和社会不同角度细腻真实的认识，大学期间，我积极参加班级和社团活动，担任班委和校羽毛球协会副会长职务。积极参与公益志愿活动，曾获得上海铁路局黄山北站优秀志愿者荣誉，中国红十字会急救员，无偿献血 1800mL，与南昌大学动物保护协会合作开发部署微信小程序：《NCU 猫咪图鉴》。研究生期间，深入学习研究业内权威著作，国际期刊和知名开源项目，跟进业内前沿动态。同时我也在不同的学习平台（Stanford，CMU 和 MIT 等国外知名大学公开课，极客时间，慕课和得到等平台）进行学习，广泛阅读（心理学，经济学，文学和医学等），希望能成为一个有趣的人。喜欢研究计算机技术的专业书籍，不断地提升自己的专业技能，从而使自己能够以专业的姿态面对问题，和善的态度对面对周围的人以达到合作共赢。

生命是一场旅程，好的坏的都是礼物。意义就是丰富而具体的体验。希望是一个好东西，也许是最好的东西。我们知道的或者相信的都不重要，重要的是我们是怎样做的。和时间做朋友，她会告诉我答案。尽人事，听天命，我们唯一稳得的就是问心无愧。保持对上天的敬畏。
